# ◆ Aditya Kumar Verma - Personal Portfolio

Welcome to the repository of my interactive spatial portfolio! This web application serves as a dynamic, canvas-driven experience detailing my professional journey, technology stack, and personal interests.

🌍 **Live Demo:** https://adityakumarverma.vercel.app/

© **Copyright:** 2026 Aditya Kumar Verma. All Rights Reserved.

## 🌟 Key Features

*   **Interactive Spatial Canvas:** A draggable, free-roam layout constructed using `framer-motion` that breaks away from traditional scrolling portfolios.
*   **Custom Vinyl Players:** Embedded mini-music players with auto-pause detection showcasing some of my favorite tracks (Dhurandhar, Bairan, and Tere Liye).
*   **Component-Driven Architecture:** Built on React and structured with modular `shadcn/ui` concepts for maintainability.
*   **Responsive Dark/Light Mode:** Full theming capabilities dynamically injected through a custom dock interface.

## 🛠️ Tech Stack

*   **Framework:** React 18 & Vite
*   **Styling:** Tailwind CSS & Lucide Icons
*   **Animation:** Framer Motion
*   **Routing:** React Router DOM

## 💻 Local Development

Clone the project and start the development server:

```sh
npm install
npm run dev
```

The application will start exclusively on `http://localhost:8080/`.
